departmentManager: plugin des départements français

Description

departmentManager est un plugin permettant la gestion des départements français.

Chaque département est associé à une région, un code (par exemple: Aude correspond à 11), un libelé et une image. La gestion se fait dans le panel d'administration de Wordpress et un affichage est possible sur toute page ou article du blog.
Pour cela, il suffit d'utiliser le shortcode [departmentShow] dans l'éditeur de Wordpress, que ce soit au sein d'une page ou d'un article.

Installation

Téléchargez le plugin, et activez le: un nouveau panel "departmentManager" est apparu dans le menu principal du panel d'administration de Wordpress.
